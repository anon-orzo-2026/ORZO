# ORZO

   Code release for the paper *ORZO: Oracle-guided Reasoning via
   Zeroth-order Optimization* (under review at NeurIPS 2026).

   The full implementation, oracle code for the four benchmarks,
   training scripts, and configurations will be released here
   within one week of the submission deadline. The repository will
   include training and evaluation code, dependency specifications,
   a results table with commands to reproduce, and configurations
   for all main-table runs.
